# Constantes_JS

## Descrição
Constantes_JS é uma aplicação educacional desenvolvida para ilustrar o conceito de constantes em JavaScript. Este projeto visa auxiliar estudantes e pesquisadores no mundo acadêmico a compreender como declarar e utilizar constantes em programas JavaScript, promovendo boas práticas de programação e imutabilidade de dados.

## Objetivos de Aprendizado
- Compreender a diferença entre variáveis e constantes em JavaScript.
- Aprender a declarar constantes usando a palavra-chave `const`.
- Explorar cenários de uso em contextos acadêmicos, como cálculos matemáticos e simulações científicas.

## Pré-requisitos
- Conhecimento básico de JavaScript.
- Node.js instalado (versão 14 ou superior).
- Um editor de código, como Visual Studio Code.

## Instalação
1. Clone o repositório:
    ```
    git clone https://github.com/seu-usuario/Constantes_JS.git
    ```
2. Navegue para o diretório do projeto:
    ```
    cd Constantes_JS
    ```
3. Instale as dependências:
    ```
    npm install
    ```

## Uso
Execute o arquivo principal para ver exemplos de constantes em ação:
```
node index.js
```

### Exemplo de Código
```javascript
const PI = 3.14159;
const GRAVITY = 9.81;

console.log(`Valor de PI: ${PI}`);
console.log(`Gravidade terrestre: ${GRAVITY} m/s²`);
```
